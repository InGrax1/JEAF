### Pendiente
- FASE 2 — Panel Administrativo Web: proyecto React + TypeScript + Vite + Tailwind CSS.
  - Autenticación JWT (login, refresh, sesión del Tesorero/Auditor).
  - Dashboard financiero: gráficas de ingresos vs gastos, desglose porcentual por categorías, flujo de caja.
  - Gestión de transacciones: DataGrid con paginación, filtros (fecha, categoría, tipo, capturista), conciliación bancaria y cancelación con motivo.
  - Módulo de configuración: CRUD de categorías (activar/desactivar), gestión de usuarios y API Keys (generar/revocar).
  - Vistas read-only para el rol Auditor.
- FASE 3 — Cierres y Reportes: descarga de reportes mensuales PDF/Excel, gráficas de comportamiento financiero.
- FASE 4 — QA, build de producción y despliegue (Vercel).
